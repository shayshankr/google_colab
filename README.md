## 🚀 LinkedIn Job Application System

A comprehensive web application to streamline your LinkedIn job search and application process.

### Features

- **🔍 Smart Job Search**: Search for jobs by keywords, location, and experience level
- **📝 Quick Apply**: Apply to multiple jobs efficiently with customizable cover letters
- **📊 Application Tracking**: Track all your job applications and search history
- **👤 Profile Management**: Manage your professional profile for consistent applications

### Getting Started

```bash
cd linkedin_jobs
pip install -r requirements.txt
python app.py
```

Then open [http://localhost:5001](http://localhost:5001) in your browser.

- Email: `demo@example.com`
- Password: `password`

## 🛍️ Handmade Shop (Legacy)

An example e-commerce website built with Flask and Stripe integration.

```bash
cd handmade_shop
pip install -r requirements.txt
python app.py
```

Then open [http://localhost:5000](http://localhost:5000) in your browser.

Replace the Stripe secret key in `app.py` with your own credentials to enable payments.
